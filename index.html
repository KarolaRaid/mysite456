Tere tulemast minu veebileheküljele X teemal!

Siin on kirjeldav lõik teksti minu veebilehe sisu kohta. Käesoleval veebilehel on teile näited kõigist varem kasutatud HTML elementidest.

Veebilehel peavad olema järgmised elemendid:
* h1 (või h2, h3, h4, h5, h6) - pealkiri
* p - lõik (paragraaf) teksti
* img (src="pildi-url-aadress.png") - pilt
* a (href="lingi-url-aadress.com") - link muule veebileheküljele
* table, tr, td - tabel, milles on vähemalt 3 rida (tr) ja igas reas vähemalt 3 lahtrit (td)

Tabeli kujundus on enda valida, aga on soovituslik kasutada kõiki varem õpitud elemente:
* Elemendi piirjooned style atribuudiga: style="border: 5px solid darkblue"
* Elemendi taustavärv: style="background-color:lightblue;"
* Tabeli sisemiste äärte eemaldamiseks: style="border-collapse:collapse;"
* Teksti värv style atribuudiga: style="color:green"
* Teksti värv font sildiga: font color="blue"

Veel huvitavat:
* br - line break, saad eraldada elemendid ka selle üksiku sildiga
* kui kasutad mitut style atribuuti, siis eralda need omavahel kooloniga ;
* värvi nime asemel saad kasutada ka HEX koodi, näiteks must on #000000 koodiga
* pildi suuruse saad defineerida width ja height atribuutidega
Küsimuste korral uuri siit vasakpoolselt ekraanilt minu koodi näidist või küsi otse!

Klikka siia, et näha erinevaid värve, mida saad HTML-s kasutada märksõnaga.
